# Жизненный цикл заказа

Текущий базовый поток:

1. Создать заказ (в зале или на доставку)
2. Подтвердить заказ
3. Отправить заказ на кухню
4. Прокручивать кухню, пока заказ не перейдёт в статус `READY` (готов)
5. Для доставки: запустить маршрут и прокручивать доставку
6. Принять оплату
7. Завершить заказ

Статусы заказа:

- `CREATED` — создан
- `CONFIRMED` — подтверждён
- `PREPARING` — подготовка
- `BAKING` — выпекание
- `READY` — готов
- `OUT_FOR_DELIVERY` — в доставке
- `COMPLETED` — выполнен
- `CANCELED` — отменён
