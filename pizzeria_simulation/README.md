# Симулятор пиццерии

CLI-симулятор работы пиццерии с чистой архитектурой и объектно-ориентированным дизайном.

## Возможности

- **Заказы** — для зала и доставка с полным жизненным циклом
- **Кухня** — выпечка пиццы с учётом вместимости печи
- **Зал** — управление столиками и посадкой гостей
- **Доставка** — назначение курьеров и отслеживание
- **Оплата** — наличные и карты, опционально чаевые
- **Сотрудники** — повара, кассиры, курьеры, менеджеры
- **Финансы** — учёт выручки и чаевых
- **Сохранение состояния** — между запусками через JSON

## Структура

```
src/pizzeria/
├── domain/          # Бизнес-логика (сущности, перечисления)
├── application/     # Use-case сервисы
├── infrastructure/  # JSON-хранилище
└── interface/       # CLI
```
